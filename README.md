## ディレクトリ構造

```html
 ルート  
　├ [css] … サイトで使用するCSSファイル   
　│　└ module.css … 見出しCSS  
　│    
　├ [framework] … サイトで使用したフレームワーク(bootstrap)のファイルを格納
　│　├ bootstrap.min.css … Bootstrapの標準CSS   
　│　└ bootstrap.min.js … Bootstrapの標準JS    
　│  
　└ index.html
 ```
